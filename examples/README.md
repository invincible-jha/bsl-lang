# Examples

| # | Example | Description |
|---|---------|-------------|
| 01 | [Quickstart](01_quickstart.py) | Parse, validate, format, and lint a BSL spec |
| 02 | [Validation](02_validation.py) | Normal and strict validation with diagnostic messages |
| 03 | [Formatting and Linting](03_formatting_linting.py) | Canonical formatting, idempotency, and lint quality checks |
| 04 | [Diff and Versioning](04_diff_versioning.py) | Compute structural diffs between BSL spec versions |
| 05 | [Schema Export](05_schema_export.py) | Export BSL specs as JSON Schema and compile to pytest stubs |
| 06 | [BslSpec Convenience](06_bslspec_convenience.py) | Programmatic API via the BslSpec convenience class |
| 07 | [Multi-Agent Specs](07_multi_agent_specs.py) | Define and validate multiple agents in a pipeline |

## Running the examples

```bash
pip install bsl-lang
python examples/01_quickstart.py
```
